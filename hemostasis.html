Гемостаз

Лабораторная оценка системы свертывания крови: скрининг, контроль терапии, дифференциальная диагностика.

АЧТВ Внутренний путь свертывания
ПТИ / PT % Внешний путь свертывания
МНО (INR) Контроль терапии варфарином
Фибриноген Конечный этап коагуляции

АЧТВ

Принцип метода: оценка внутреннего и общего пути свертывания (XII, XI, IX, VIII, X, V, II, I).

Референс: 24–35 секунд (зависит от реагента).

Изолированное удлинение АЧТВ — алгоритм

  1. Проверить преаналитику (цитрат, гемолиз, сгусток).
  2. Исключить гепарин (анамнез, анти-Xa).
  3. Смесь 1:1 с нормальной плазмой:
    • Коррекция → дефицит факторов (VIII, IX, XI, XII).
    • Нет коррекции → ингибитор.
  4. При подозрении — тесты на волчаночный антикоагулянт.

ПТИ / PT %

Принцип: оценка внешнего пути свертывания (VII, X, V, II, I).

Референс: 70–120 %.

Печень vs варфарин

  • Поражение печени: ↓ПТИ, ↓фибриноген, ↑АЧТВ.
  • Варфарин: ↓ПТИ, ↑МНО, АЧТВ чаще в норме.

МНО (INR)

Принцип: стандартизированная форма PT для мониторинга терапии антагонистами витамина K.

Терапевтические диапазоны

  • Большинство показаний: 2.0–3.0
  • Механические клапаны: 2.5–3.5

Ошибки и ловушки

  • Нарушение соотношения кровь / цитрат.
  • Высокий гематокрит.
  • Различия ISI тромбопластинов.

Клинические алгоритмы

  • ↑АЧТВ + нормальный PT → внутренний путь.
  • ↑PT / МНО + нормальный АЧТВ → внешний путь / варфарин.
  • ↑АЧТВ + ↑PT → печень, ДВС, дефицит витамина K.
